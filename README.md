# これは何？
2023年度版ポートフォリオサイトのソースコードリポジトリです。

https://karakure178.com/

## 使用技術(メイン)
- SCSS
- Vue3
- Gsap
- moveto
- Swiper
- pinia
- p5.js


## 実行コマンド
| Command                   | Action                                           |
| :------------------------ | :----------------------------------------------- |
| `npm install`             | 環境のインストール                            |
| `npm ci`             | バージョンを固定してインストール                            |
| `npm start`             | サーバーの立ち上げ：`localhost:3001`      |
